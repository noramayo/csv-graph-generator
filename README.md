# 📊 CSV グラフ自動生成アプリ

CSVファイルをアップロードするだけで、棒グラフ・折れ線グラフ・円グラフを自動生成するWebアプリです。
インストール不要。ブラウザで `index.html` を開くだけで動きます。

> このリポジトリは note 記事「**ExcelのデータをCSVで貼るだけで、グラフが自動完成するWebアプリをClaude Codeで作った【プロンプト全文・GitHub公開】**」の公開コードです。
> 記事はこちら → [note リンクをここに貼る]

---

## ✨ できること

- CSVファイルをドラッグ＆ドロップするだけでグラフが完成
- **棒グラフ・折れ線グラフ・円グラフ**の3種類をワンクリックで切り替え
- グラフのタイトルを自由に変更できる
- グラフを **PNG画像として保存**できる
- 読み込んだデータのプレビューテーブルを表示
- Shift-JIS / UTF-8 どちらのCSVにも対応（日本語OK）
- スマートフォン・タブレットでも動作

---

## 🚀 使い方

**ステップ1：** このページ右上の緑のボタン「Code」→「Download ZIP」でダウンロード

**ステップ2：** ZIPを解凍して `index.html` をダブルクリック

**ステップ3：** ブラウザが開いたらCSVファイルをドラッグ＆ドロップ

**ステップ4：** グラフが自動表示されます。種類の切り替えや保存もその場でできます。

## 📋 CSVのフォーマット

1行目をヘッダー（列名）、2行目以降をデータとして自動認識します。

```csv
月,売上,コスト
1月,120,80
2月,150,90
3月,130,85
4月,180,100
```

- **1列目**：X軸のラベル（月名・商品名・担当者名など）
- **2列目以降**：グラフにする数値データ（複数列OK）
- 数値以外の列は自動的に除外されます

## 🔧 カスタマイズしたい方へ

このアプリはすべて `index.html` の1ファイルに収まっています。
Claude Code（またはChatGPT）に以下のように伝えるだけで改造できます。

**色を変えたい場合：**
```
index.html のグラフの色を会社のブランドカラー（#FF6B35）に変えてください
```

**グラフの種類を追加したい場合：**
```
index.html に散布図を追加してください。既存の棒・折れ線・円と同じタブで切り替えられるようにしてください
```

**Excel形式（.xlsx）にも対応させたい場合：**
```
index.html をCSVだけでなく.xlsxファイルにも対応させてください
```

改善のプルリクエストも歓迎します！
